# e_cash_app

An e-Cash app created using Dart Language with Flutter SDK.

This app demonstrates the Widgets:
- Slivers
- Opacity
- Drawer
- Shadermask
- Stack
- CarouselSlider
- DataTable

Disclaimer: The assets was used for educational purposes only.
